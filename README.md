# 멜로플레이어

멜론차트 TOP 50에 있는 노래들을 크롤링한 뒤, 유튜브를 통해 다운로드하고 다운로드 된 노래를 플레이어를 통해 들을 수 있습니다.

## 멜로플레이어 셋업

### 준비물

```
python3
```

### 설치

1. Github 레포지토리 클론
    ```bash
    $ git clone https://github.com/legitcode267/meloplayer
    ```
### 실행

1. 앱 실행
    ```bash
    $ cd meloplayer
    $ python main.py
    ```
   
앱 실행에 오랜 시간이 걸릴 수 있습니다.
